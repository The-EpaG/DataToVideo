# DataToVideo

A simple Go project to convert a file to a video.

## How to build it

DataToVideo can be build with make if you have go installed

```bash
make
```

## How to run it

DataToVideo can be run with make if you have go installed

```bash
./bin/DataToVideo
```

## How to run it without build it

DataToVideo can be run with make if you have go installed

```bash
make run
```
